## Learning Git

``` 
## initialise git repo
git init

# add to tracking area
 git add .
# or your can add single file by 
git add filename

# commit your code 
git commit -m "my message"

#push into git from local folder
git push -u origin
```

## adding remote
```
git remote add origin git@github.com:supii95/first-repo.git
git branch -M main
git push -u origin main
```
